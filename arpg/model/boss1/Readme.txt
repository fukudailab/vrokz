組み立て式のステージです。


◆使い方

1.　透過材質があるステージなので、スカイドームを入れたい方は
　　事前に設定しておいて下さい

2.　パーツを数字順に読み込んで下さい。
　　02滝はテクスチャスクロールのエフェクト付きパーツです
　　xファイルだけを読み込んで下さい。

3.　04は背景だけ使いたい場合は読み込まなくてOK。
　　使用する場合はAutoLuminous併用推奨です。
　　お好みでtr値を下げると背景に馴染みます。

4.　PCスペック的に可能であれば、
　　ぼかし系のエフェクトで背景を適度にぼかして下さい。

5.　さらに可能であれば水面エフェクト併用推奨です。
　　おすすめはビームマンPの水面エフェクトver5.0／穏やかな川
　　Y＝0.5／Ry＝180／Si＝150ぐらいで。調節後の登録を忘れずに！


エフェクトはFuria様のMMEで動く噴水を改変した物です。


改造・パーツ使用自由

MMD用再配布の場合は当方まで配布先を連絡して下されば可とします。
（blogのコメント・拍手・twitter等連絡手段は問いません）

その他での用途は個人使用ならOK。再配布は不可。

基本、商用利用は不可ですが、
MMD使用を前提としている映像作品等には個別に許可を出しています。
